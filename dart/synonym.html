  • 最新说明: https:/www.dartlang.org/resources/synonyms

// 使用下面两个标签可以支持所有浏览器

// 如果浏览器不支持 Dart VM
// 则下面的脚本会载入 javascript 版本的
// 程序代码

queue.add('event:49309');

var eventId = queue.removeFirst();

print(eventId == 'event:32342'); // true
