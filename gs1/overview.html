# GS1是全球商业语言

GS1标准经过40多年发展起来的。点这里了解历史

它提供了一种通用语言，并有助于创建无缝的工作流程，使企业可以在全球范围内以相同的方式识别，获得和共享信息。

通过GS1标准的强大功能，组织可以使用准确，可靠的信息，从而可以在正确的时间将正确的产品放置在正确的位置，并允许消费者找到他们想要的产品。

可以将GS1标准视为产品的数据“DNA”，这是将项目与所有有关该项目的信息联系起来的基础。在这个基础上，大小公司都在建立更好的业务方式。例如，GS1标准启用：

  • 消费者跟踪他们所吃的食物-从农场到餐桌
  • 购物者可以从其移动设备访问详细的产品信息
  • 公司通过提高其供应链中的产品和物料可见性来节省成本

# GS1标准概览：识别，获取，共享

# 识别

条形码后面的数字是GS1标识。GS1标识号在物理和数字世界中都可以使用，可以唯一地区分产品，物流单位，位置，资产，文件以及整个供应链（从制造商到消费者）之间的关系。 GS1有各种标识

识别内容 条码
Companies 公司实体 GS1 Company Prefix
Products 产品 GTIN®
Locations 位置 GLN
Logistics Units 物流单元 SSCC
Documents 文件 GDTI
Service Relationships 服务关系 GSRN
Assets 资产 GIAI, GRAI
Shipments 货运 GSIN, GINC

# 获取

GS1条形码可以保存有关产品或物品的数据。不同的条形码具有不同的用途，具体取决于要扫描的位置以及需要保存的数据类型。

获取内容 条码类型
Product Data for Point of Sale POS用产品数据 U.P.C. / EAN
Product Data for Point of Care/Use POC用产品数据 GS1 DataMatrix, GS1 DataBar®
Coupons 优惠券 GS1 DataBar
Cartons, Cases, Pallets 纸箱，箱子，托盘 ITF-14, GS1-128
Detailed Product Data and Marketing 详细产品数据和市场 GS1 QR Code
Product Data Using RFID 使用RFID的产品数据 EPC®-Enabled RFID

# 共享

GS1具有用于以电子方式共享产品和位置信息的标准，从而无需手动重新键入密钥，并提高了业务通信的准确性和效率。

|共享内容|方式| |Master Product Data 产品主数据| GDSN®| |Business Documents, Messages 商业文件、消息| EDI, XML| |Business Messages for high-tech 电子商务| RosettaNet®| |Product Event Data 产品事件数据|EPCIS |

# GS1标准制定

GS1全球办公室使用全球标准管理流程（GSMP）制定国际标准。GSMP是文档开发生命周期，其中包括一套全面的规则，使GS1的利益相关者社区可以就用户驱动的标准达成共识。要访问标准制定网站，请单击此处。

##GSMP标准制定工作组 GSMP汇集了来自世界各地的用户，以制定GSMP工作组中的供应链标准。这些团体在GSMP下运作，并遵守反托拉斯法。要参与工作组，请单击此处。

# UNSPSC®

由GS1 US管理的联合国标准产品和服务法规®是一种开放的，全球性的，跨行业的标准，用于对产品和服务进行有效，准确的分类。要查看UNSPSC ®，请点击这里。
